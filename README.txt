Hand Drawing Project - Setup Instructions

FIRST STEP:
Download python version 3.9 in your device.
Mac: https://www.python.org/ftp/python/3.9.6/python-3.9.6-macos11.pkg
Windows: https://www.python.org/ftp/python/3.9.6/python-3.9.6-amd64.exe
(For Windows) Make sure to check the "Add Python to PATH" checkbox when installing Python.

FOR WINDOWS USERS:
1. Download the ZIP file from GitHub and extract it.
2. Double-click "run_project.bat"
3. Follow the instructions on screen

FOR MAC USERS:
1. Download the ZIP file from GitHub and extract it.
2. Open Terminal in the project folder > run "python3.9 run_project_mac.py"
3. Follow the instructions on screen

For help, contact your workshop instructor!
