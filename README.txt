Hand Drawing Project - Setup Instructions

FIRST STEP:
Download python version 3.9 in your device.
Mac: https://www.python.org/ftp/python/3.9.6/python-3.9.6-macos11.pkg
Windows: https://www.python.org/ftp/python/3.9.6/python-3.9.6-amd64.exe
Make sure to check the "Add Python to PATH" checkbox (if any) when installing Python.

FOR WINDOWS USERS:
1. Download the zip files from GitHub and extract it.
2. Double-click "run_project.bat"
3. Follow the instructions on screen

FOR MAC USERS:
1. Download the zip files from GitHub and extract it.
2. Go to System Settings > Privacy & Security > scroll down > press "Open Anyways".
3. First time only: open Terminal in the project folder > run "chmod +x run_project.command"
4. Then double-click "run_project.command" file.
5. Follow the instructions on screen

For help, contact your workshop instructor!
