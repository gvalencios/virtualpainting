Hand Drawing Project - Setup Instructions

FOR WINDOWS USERS:
1. Download all files to a folder
2. Double-click "run_project.bat"
3. Follow the instructions on screen

FOR MAC USERS:
1. Download all files to a folder
2. First time only: in Terminal run "chmod +x run_project.command"
3. Then double-click "run_project.command"
4. Follow the instructions on screen

REQUIRED FILES (must all be in same folder):
- run_project.bat (Windows)
- run_project.command (Mac)
- HandTrackingModule.py
- virtual_painter.py
- README.txt (this file)

TROUBLESHOOTING:
- Make sure Python is installed from python.org
- Allow camera access when prompted
- Ensure all files are in the same folder
- On Mac, if you see "cannot be opened" warning:
    Right-click -> Open -> Click "Open" anyway

For help, contact your workshop instructor!
